Abstract

อุปกรณ์ติดตามคุณภาพอากาศ” จัดทำขึ้นโดยมีวัตถุประสงค์เพื่อพัฒนาเครื่องติดตามคุณภาพอากาศที่สามารถตรวจวัดค่าฝุ่นละอองในอากาศ ได้แก่ PM10, PM2.5 และ PM1.0 ในหน่วยไมโครกรัมต่อลูกบาศก์เมตร รวมถึงอุณหภูมิ (°C) และความชื้นสัมพัทธ์ (%) ได้แบบเรียลไทม์ โดยใช้อุปกรณ์เซนเซอร์ DHT22 สำหรับวัดอุณหภูมิและความชื้น และเซนเซอร์ PMS3003 สำหรับตรวจวัดค่าฝุ่นละออง ข้อมูลจากเซนเซอร์จะถูกประมวลผลโดยไมโครคอนโทรลเลอร์ Arduino UNO R4 WiFi ก่อนส่งต่อไปยังระบบแสดงผลหรือเว็บไซต์เพื่อให้ผู้ใช้สามารถติดตามข้อมูลได้จากระยะไกล ระบบดังกล่าวช่วยให้ผู้ใช้งานสามารถเฝ้าระวังคุณภาพอากาศในพื้นที่ได้อย่างสะดวก แม่นยำ และไม่จำเป็นต้องเข้าไปในพื้นที่ด้วยตนเอง

Components

Temperature & Humidity Sensor - ตรวจวัดอุณหภูมิและความชื้น

Dust Sensor - ตรวจวัดค่าฝุ่น

Arduino Board - ใช้เป็นสมองหลักในการควบคุมระบบ

DHT22

เซนเซอร์ DHT22 เป็นอุปกรณ์ตรวจวัด อุณหภูมิและความชื้นสัมพัทธ์ แบบดิจิทัล ทำงานโดยใช้ Thermistor สำหรับวัดอุณหภูมิ และ Capacitive Humidity Sensor สำหรับวัดความชื้น ตัวเซนเซอร์จะทำการแปลงค่าที่วัดได้เป็นสัญญาณดิจิทัล DHT22 มีความละเอียดในการวัดความชื้น ±2–5% RH และอุณหภูมิ ±0.5°C ทำให้เหมาะสำหรับการใช้งานที่ต้องการความแม่นยำในระดับปานกลางถึงสูง

PMS3003

เซนเซอร์ PMS3003 เป็นอุปกรณ์ตรวจวัดค่าฝุ่นละอองในอากาศแบบ เลเซอร์ (Laser Scattering Sensor) โดยใช้หลักการการกระเจิงของแสง (Light Scattering Principle) ภายในโมดูลจะมี แหล่งกำเนิดแสงเลเซอร์ และ ตัวรับแสง (Photodiode) เมื่ออากาศที่มีฝุ่นละอองถูกดูดผ่านเข้าไปภายในเซนเซอร์โดยพัดลมขนาดเล็ก ฝุ่นละอองจะสะท้อนแสงเลเซอร์ในมุมต่าง ๆ ความเข้มของแสงที่สะท้อนกลับจะถูกวัดและนำไปประมวลผลเพื่อคำนวณปริมาณฝุ่นในอากาศ หน่วยเป็นไมโครกรัมต่อลูกบาศก์เมตร (µg/m³) โดยสามารถตรวจวัดค่าฝุ่นได้หลายขนาด เช่น PM1.0, PM2.5 และ PM10 ค่าเหล่านี้ถูกส่งออกในรูปแบบสัญญาณดิจิทัลไปยังไมโครคอนโทรลเลอร์เพื่อแสดงผลหรือประมวลผลต่อไป

Arduino R4 WiFi

Arduino R4 Wi-Fi คือบอร์ดไมโครคอนโทรลเลอร์รุ่นพัฒนาของ Arduino Uno ที่มีความสามารถในการเชื่อมต่อไร้สาย (Wi-Fi, Bluetooth) และมีประสิทธิภาพสูงขึ้นจากไมโครโปรเซสเซอร์แบบ 32 บิต Renesas RA4M1 บอร์ดนี้ยังมาพร้อมกับฟีเจอร์ใหม่ๆ ที่ช่วยให้การพัฒนาโปรเจกต์ทำได้ง่ายขึ้น เช่น จอ LED Matrix ขนาด 12x8, ตัวเชื่อมต่อ Qwiic I2C, และฟังก์ชันสำหรับการตรวจสอบและควบคุมโปรเจกต์จากระยะไกลผ่าน Arduino IoT Cloud
